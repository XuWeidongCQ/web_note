# 三、单例模式

* 一个构造函数只允许被实例化一次

```js
class SingleCase {
  constructor(){}
}

const getSingleInstance = (function(){
  let instance = null
  return function(...args){
    instance = instance ? instance : new SingleCase(...args)
    return instance
  }
})()

let one = getSingleInstance()
```
